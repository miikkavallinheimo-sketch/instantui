## OpenAI Trend & Vibe Generation

Two scripts use OpenAI's **Responses API**:

- `scripts/updateTrends.ts`:

  - Uses `tools: [{ type: "web_search" }]` to research current (2025â€“2026) font and color trends.
  - Writes the result into `src/data/trends.json`.

- `scripts/generateVibesFromTrends.ts`:

  - Reads `src/data/trends.json`.
  - Asks OpenAI to generate new **vibe suggestions** (names, descriptions, recommended fonts and colors).
  - Writes to `src/data/generatedVibes.json`.

### Run them manually

```bash
npm run update:trends
npm run generate:vibes
```

Typically you'd wire these into a **GitHub Action** with a cron schedule to refresh trends and generated vibes weekly.
